> use vue2.0 & vuex & vue-router to build Single Page Web Application.


# Usage

`npm install`

`npm run dev`

then visit <http://localhost:3000> you will get it.

![image](webapp/static/images/QQ20161015-0.jpg)


# Production

 `npm run prod`
